Comandos Staff - RagnarMC

Guía de Screen Share

1. AnyDesk: Desactivar conexiones directas para no exponer IP.

2. Everything: Buscar grabadores: .mp4|.avi|.mkv|.webm|.anydesk

3. System Informer: Revisar procesos y strings sospechosos (file:///, pcaclient).

4. Directorios importantes: PCA General DataBase, Windows History, CrashDumps, ReportArchive.

5. WinPrefetchView: Ver programas ejecutados recientemente.

6. USBDeview: Historial de dispositivos USB conectados.

7. Task Manager: Finalizar navegadores para evitar grabaciones.


Mantén profesionalismo y revisa solo carpetas relacionadas con Minecraft.
